
# Django Book CRUD with Docker - Made with ❤️ for learning

A clean and simple **Django** web application for managing books with full CRUD functionality. Designed as a beginner-friendly project to practice **Django**, **Forms**, **Templates**, and **Docker** skills.

![Python](https://img.shields.io/badge/Python-3.12-blue)
![Django](https://img.shields.io/badge/Django-5.1-green)
![Docker](https://img.shields.io/badge/Docker-✅-blue)
![PostgreSQL](https://img.shields.io/badge/PostgreSQL-✅-blue)

## Features

- Add, View, Edit, and Delete books
- Responsive and clean UI
- Form validation
- Docker + Docker Compose support
- PostgreSQL database
- Ready for development and learning

## ech Stack

- **Backend**: Django 5.1
- **Database**: PostgreSQL
- **Frontend**: HTML + CSS
- **Containerization**: Docker & Docker Compose
- **Web Server**: Gunicorn (production ready)

# 2. Start the application
